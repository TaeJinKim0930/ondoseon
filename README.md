# 온도썬 🌡️

썸~연애초반 20대를 위한 커플 궁합 테스트. 양자택일 20문항으로 가치관 궁합을 재고, 두 사람의 답을 비교해 "잘 맞는 점 / 부딪힐 점"을 점수가 아닌 **온도**로 알려줍니다. 백엔드 없는 정적 웹.

## 실행
- **로컬**: `index.html`을 더블클릭하면 바로 열립니다(일반 스크립트라 file://에서도 동작).
- **GitHub Pages 배포**: 이 폴더를 repo에 올리고 Settings → Pages → Branch `main` / root 선택.

## 동작 방식
- 답안은 24bit(20문항 + MBTI 4축)로 인코딩되어 **URL 안에만** 담깁니다(서버 저장 0건).
- A가 풀고 `?p=토큰` 링크 공유 → B가 풀면 `?p=...&q=...` 공동 결과. 누가 열어도 같은 결과(결정론).
- MBTI는 점수에 반영되지 않는 **해석 레이어**입니다(미입력 시 생략).

## 구조
```
index.html        화면 셸(랜딩/퀴즈/MBTI/솔로/초대/결과/에러)
css/styles.css    디자인 시스템(라이트 파스텔, 온도 은유)
js/data.js        문항 + 카피(단일 진실원: 문항 category가 카테고리 귀속 결정)
js/codec.js       답안 ↔ URL 토큰 (base64url, v1/v1n prefix)
js/engine.js      점수 엔진(호환매트릭스→카테고리 가중평균→게이트 캡→밴드) + 자기검증
js/report.js      공동 결과 DOM 렌더
js/capture.js     결과 카드 PNG 캡처(순수 canvas)
js/app.js         라우팅/퀴즈/공유 컨트롤러
```

## 검증
`engine.js`의 `runSelfTest()`가 페이지 로드 시 콘솔에서 골든 케이스(전부일치=100 / 전부불일치 / 게이트 캡=69)를 자동 검증합니다.

> 재미·자기이해용입니다. 점수가 관계의 성공을 보장하지 않으며, 정해진 '천생연분'도 '최악 궁합'도 없습니다.
